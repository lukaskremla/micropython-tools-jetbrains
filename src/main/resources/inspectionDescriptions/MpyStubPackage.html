Detects when no stub package is selected while type hints are enabled.

This inspection helps identify projects where type hints are enabled but no valid stub package has been configured. Without a selected stub package, essential IDE features such as autocompletion, type checking, and documentation lookup will not function for MicroPython-specific APIs.

To fix this, go to MicroPython Tools Settings and select or install a stub package that matches your board and MicroPython version.

This issue is commonly encountered after project import or first-time plugin setup.
